*Amazon Scraper 

Script en Python que realiza scraping de productos en Amazon usando Selenium.

*Funcionalidades

- Búsqueda automática de productos
- Extracción de:
  - Nombre
  - Precio
  - Descuento (%)
  - Calificación
  - Tipo (Orgánico / Patrocinado)
  - Estado (Disponible / Agotado)
- Navegación por múltiples páginas
- Exportación a Excel segmentado

* Requisitos

- Python 3.14.3
- Google Chrome
- ChromeDriver

Instalar dependencias:
pip install -r requirements.txt

* Ejecución

```bash
python main.py
